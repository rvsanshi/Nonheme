# Nonheme
A specialized dataset containing the two-dimensional structures of non-heme iron complex molecules and their corresponding diabatic bond dissociation energy (BDE) values, as well as model training for predicting diabatic BDE values.
There are 668 two-dimensional topological structures of non-heme iron complexes, all of which are saved in SDF files.
There are a total of 917 corresponding diabatic BDE values, all of which are saved in the BDE.csv file.
The machine learning code is saved in test.ipynb.
